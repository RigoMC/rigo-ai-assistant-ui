# 🧠 Rigo AI Assistant UI

Interfaz de usuario para un asistente de inteligencia artificial, construida con React, Next.js y TailwindCSS. 

---

## 🚀 Características

- Integración con API de IA (OpenAI)
- Diseño modular
- Estilos personalizados con TailwindCSS

---

## 🛠️ Tecnologías

- **React** + **Next.js**
- **TailwindCSS**
- **TypeScript**
